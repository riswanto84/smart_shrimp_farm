# Perbaikan Data Sampling

- Data sampling diurutkan berdasarkan tanggal terbaru, kemudian ID terbaru.
- Nilai desimal pada tabel, KPI, halaman detail, PDF, dan Excel ditampilkan tepat 3 angka di belakang koma.
- Kolom jumlah ekor/populasi tetap bilangan bulat.
- Tidak ada perubahan struktur database.
