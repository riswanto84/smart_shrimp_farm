# Perbaikan Rekonsiliasi Modal Awal

Perubahan:
- Modal Pemilik pada menu Saldo Awal tidak lagi dihitung menggunakan laba/rugi tahun berjalan.
- Rumus rekonsiliasi: Total Aset Awal - Total Kewajiban Awal - Laba Ditahan.
- Piutang, utang usaha, aset tetap bruto, dan akumulasi penyusutan diambil dari modul terkait.
- Ditambahkan tombol Rekonsiliasi Modal Awal dan pratinjau angka yang disarankan.
- Neraca berjalan tetap memasukkan laba/rugi tahun berjalan sebagai komponen ekuitas.
- Tidak memerlukan migrasi database.
