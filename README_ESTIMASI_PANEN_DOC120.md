# Estimasi Panen DOC 120

Dashboard utama kini menampilkan proyeksi per kolam dan total panen pada DOC 120.

Rumus:
- Sisa hari = 120 - DOC sampling terakhir
- ABW DOC 120 = ABW Today + (ADG Actual x sisa hari)
- Size DOC 120 = 1000 / ABW DOC 120
- Biomassa DOC 120 = Populasi FR x ABW DOC 120 / 1000

Ringkasan total:
- Proyeksi teoritis: 100%
- Skenario normal: 95%
- Skenario konservatif: 90%

Asumsi: ADG tetap, populasi FR tidak berkurang, dan tidak ada panen parsial sampai DOC 120.
