# Perbaikan Rumus ADG Actual

ADG Weekly Actual sekarang dihitung konsisten dengan rumus:

```text
(ABW Today - ABW Last) / 7
```

Contoh K2: `(8,75 - 7,03) / 7 = 0,246`, ditampilkan menjadi `0,25 gr/hari`.

Perubahan berlaku pada input manual, proses import Excel, tabel Data Sampling, dan kartu rata-rata ADG. Import ulang dengan mode **Perbarui data lama** untuk memperbaiki record yang telah tersimpan.
