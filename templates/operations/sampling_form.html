Mapping Rumus Excel

ABW Today = Berat SHRIMP / Jumlah SHRIMP.

Size = 1000 / ABW Today.

ADG Actual = (ABW Today - ABW Last) / interval hari.

ADG Accum = ABW Today / DOC.

Biomassa FR = F/D / FR × 100.

Populasi FR = Biomassa FR × Size.

SR FR = Populasi FR / Tebar × 100.

Biomassa Index = Populasi Index / Size.

FCR = Pakan Kumulatif / Biomassa FR.

Catatan: ABW Last, F/D, Pakan Kumulatif, dan Tebar akan otomatis diisi dari data sebelumnya jika tersedia.
