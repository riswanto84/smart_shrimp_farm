# Perbaikan Dashboard Produksi dan Tombol Aksi

- FCR dan biomassa dashboard menggunakan nilai tersimpan, lalu fallback perhitungan dari populasi, ABW, dan pakan kumulatif.
- Data lama/import yang belum menyimpan field hasil turunan tetap dapat ditampilkan.
- Tombol Detail, Edit, dan Hapus pada Cek Anco, Sampling, dan Siphon hanya menampilkan ikon.
- Tooltip, title, dan aria-label tetap tersedia.
