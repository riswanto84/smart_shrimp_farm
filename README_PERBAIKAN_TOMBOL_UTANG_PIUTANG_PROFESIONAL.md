# Perbaikan Tombol Utang dan Piutang

Halaman Pengeluaran Operasional diperbarui dengan tombol navigasi Utang Usaha dan Piutang Usaha bergaya profesional.

Perubahan:
- Bentuk action card ringkas dengan ikon dan deskripsi.
- Warna Utang dan Piutang dibedakan secara halus.
- Hover, shadow, dan border yang konsisten dengan Smart Shrimp Farm.
- Tombol Tambah Pengeluaran tetap menjadi aksi utama.
- Tata letak responsif untuk desktop, tablet, dan mobile.

Tidak ada perubahan model atau migrasi database.
