# Perbaikan Panel Kanan Edit Nota (Force Flex)

- Mengganti layout grid menjadi flex row tanpa wrapping.
- Menetapkan lebar panel ringkasan 320 px.
- Menetapkan lebar minimum area form 620 px.
- Layout desktop minimum 980 px sehingga panel tidak dapat jatuh ke bawah akibat CSS lain atau zoom domain.
- Pada perangkat sangat kecil (maksimal 520 px), panel kembali ditampilkan di bawah.
- Tidak memerlukan migrasi database.
