# Laporan Akhir Siklus untuk Semua Role

- Halaman daftar siklus dapat dibuka oleh seluruh pengguna yang sudah login.
- Tambah dan edit siklus tetap khusus Owner.
- Tombol PDF hanya muncul untuk siklus berstatus Selesai.
- Endpoint PDF dilindungi login, tetapi tidak dibatasi role.
- Laporan memuat periode, KPI snapshot akhir, biomassa, ABW, ADG, FCR, panen, pakan, mortalitas, jumlah anco/parameter, dan sampling terakhir per kolam.
- Tidak ada perubahan model atau migrasi database.
