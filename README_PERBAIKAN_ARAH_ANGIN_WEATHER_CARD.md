# Perbaikan Arah Angin Weather Card

- Menghapus singkatan satu huruf seperti `T`.
- Menampilkan arah angin dalam Bahasa Indonesia, misalnya `→ Timur` atau `↗ Timur Laut`.
- Menambahkan tooltip berisi arah dan derajat angin aktual dari Open-Meteo.
- Jika arah angin tidak tersedia, tampil `Arah tidak tersedia`.
- Tidak memerlukan migrasi database.
