# Perbaikan UI/UX AI Prediksi Panen Parsial

Halaman AI Prediksi Panen Parsial diperbarui menjadi tampilan profesional dan responsif.

Perubahan:
- Hero header dengan status Ollama, model, dan endpoint.
- Form target panen dalam satu panel yang rapi.
- Input Target Size manual angka bulat dengan default 30.
- KPI tanggal, size, biomassa, dan potensi nilai dengan hierarchy visual lebih jelas.
- Ringkasan data sampling dalam metric cards.
- Skenario optimistis, normal, dan konservatif.
- Rekomendasi panen dalam format action cards.
- Tampilan responsif untuk desktop, tablet, dan ponsel.

Tidak ada perubahan model atau database, sehingga migrasi tidak diperlukan.
