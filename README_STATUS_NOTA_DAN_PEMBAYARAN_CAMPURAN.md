# Perbaikan Status Nota dan Pembayaran Campuran

Perubahan:
- Opsi Pembayaran Campuran tetap tersedia pada Kasir dan Edit Nota.
- Halaman Edit Nota memiliki pilihan Status Pembayaran: Lunas atau Belum Lunas.
- Status dapat diubah manual tanpa menghapus rincian metode pembayaran campuran.
- Upload bukti/tanda terima tetap dipertahankan.

Tidak ada migrasi database baru karena field dan pilihan Pembayaran Campuran sudah tersedia pada migrasi sales 0007.
